# سيرفر مشاهدة يوتيوب مع بعض (Watch Party)

هذا سيرفر منفصل عن بوت تيليغرام، وظيفته مزامنة تشغيل فيديوهات يوتيوب
بين عدة أشخاص بنفس الغرفة (نفس كود الغرفة المستخدم بالبوت).

## ⚠️ مهم: هذا السيرفر لازم يشتغل على استضافة حقيقية على الإنترنت
ما ينفع تشغله على جهازك الشخصي فقط، لأن أصحابك من أجهزة/شبكات ثانية
لازم يقدروا يوصلوا له عبر رابط عام (HTTPS).

## النشر على Render (مجاني، الأسهل للمبتدئين)

1. أنشئ حساب مجاني على https://render.com
2. من لوحة التحكم، اختر **New +** ثم **Web Service**
3. اختار **Deploy from a public Git repository** (تحتاج ترفع هذا المجلد
   لحساب GitHub أول)، أو اسأل المطوّر لو تبي طريقة رفع بديلة بدون Git
4. عند الإعداد:
   - **Runtime**: Python 3
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `uvicorn server:app --host 0.0.0.0 --port $PORT`
5. بعد النشر، Render بيعطيك رابط شكله:
   `https://اسم-مشروعك.onrender.com`
6. خذ هذا الرابط واستخدمه كمتغير بيئة `WATCH_PARTY_URL` عند تشغيل بوت
   تيليغرام (بمجلد telegram_group_bot):
   ```powershell
   $env:WATCH_PARTY_URL="https://اسم-مشروعك.onrender.com"
   ```

## التجربة محلياً (بدون نشر، فقط للتأكد إن الكود شغال على جهازك)

```powershell
pip install -r requirements.txt
uvicorn server:app --reload
```
بعدها افتح المتصفح على: http://127.0.0.1:8000/?room=TEST
(هذا يشتغل بس على جهازك، صديقك ما يقدر يدخل عليه من بيته)

## ملاحظة عن الخطط المجانية
بعض الاستضافات المجانية (زي Render) توقف السيرفر مؤقتاً لو ما فيه زيارات
لفترة، وياخذ وقت (10-30 ثانية) يرجع يشتغل أول ما حد يفتح الرابط. هذا طبيعي
بالخطة المجانية، ولو أزعجك مستقبلاً نقدر نطور لخطة مدفوعة رخيصة تخلي
السيرفر شغال دايماً.
